Les CSS (Cascading Style Sheets en anglais, ou « feuilles de style en cascade ») sont le code utilisé pour mettre en forme une page web

Il est possible d’utiliser des styles en les définissant :

  • dans un fichier CSS (la norme et le plus utilisé)
  • dans la page HTML, dans la balise <head> suivi de la valise <style> (rare, utilisation essentiellement pédagogique car dans le même fichier que l’HTML)
  • de façon exceptionnelle dans une balise pour forcer le style grâce à l’attribut style
