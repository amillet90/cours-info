EXERCICE : créer les 3 tables ci dessous :

  • Créer une table TD_telephone, cette table se compose des champs suivants :

    • un identifiant (unique non null) par un entier qui s’incrémente automatiquement (gérer par MYSQL) de nom idTelephone
    • un champ type_id qui fait référence au type du téléphone, avec une chaîne de 2 caractères
    • un champ marque_id qui fait référence à la marque du téléphone, de type entier
    • un champ date_achat qui indique la date d’achat du produit de type date
    • un champ prix qui permet de saisir un numérique de valeur maximum 9999999,99
    • un champ proprietaire_id qui référence au propriétaire, de type entier
    • un champ couleur de type chaîne de caractère variable de taille maximum de 25 caractères

  • Créer une table TD_type_tel, cette table se compose des champs suivants :

    • un identifiant (unique non NULL) avec une chaîne de 2 caractères de nom idType
    • un champ libelle de type chaîne de caractère variable de taille maximum de 25 caractères

    • un champ pays de type chaîne de caractère variable de taille maximum de 25 caractères
